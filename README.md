## Описание
Приложение на Android, предназначенное для подключения смартфона к видеорегистратору с Wi-Fi модулем.

Иногда при обгоне крупногабаритных транспортных средств на трассе бывает небезопасно "выглядывать" на встречную полосу, чтобы удостовериться в отсутствии автомобилей. 
Во избежание возможных опасных ситуаций, проект "Lorry Vision" предлагает устанавливать на фуры и другие крупные транспортные средства видеорегистратор с Wi-Fi передатчиком,
который будет передавать сигнал с видеоизображением на едущие сзади автомобили. Чтобы избежать больших затрат на создание и распространение специальных устройств для передачи-приема
сигнала Wi-fi, в качестве специальных устройств выбраны видеорегистратор с Wi-Fi модулем для установки в фурах и смартфон на операционной системе Android для приема видеосигнала 
в обгоняющем автомобиле.

## Требования к приложению
Основное требование - обеспечение безопасности использования приложения. Например, если водитель один в машине, приложение должно работать максимально автономно.
- Автоматическое подключение к Wi-Fi сети любого видеорегистратора
- Имя и пароль Wi-Fi сети видеорегистратора должны соответствовать специальным требованиям для автоматической 
  аутентификации в приложении: это обеспечит защиту от несанкционированного подключения к сети
- Отключение от Wi-Fi сети после обгона
- Получение видеоизображения (без звука)
- Работа в фоновом режиме
- Управление видеорегистратором должно быть запрещено
